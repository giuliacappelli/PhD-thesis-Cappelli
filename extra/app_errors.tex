\chapter{Squared errors: distance between actual judgments and values predicted by the model}
\labapp{app_errors}

This appendix collects the actual acceptability judgments provided by human participants, the values predicted by my Stochastic Optimality Theoretic model (in the full version using Behavioral PISA, as explained in full in \refch{model}), and the squared error for each sentence in the set of stimuli for English and Italian (which the interested reader can peruse in \refapp{app_stimuli}).\\
These data are also available \href{https://github.com/giuliacappelli/dissertationData}{here on my GitHub profile}\sidenote{https://github.com/giuliacappelli/ dissertationData}, while the Python scripts I wrote to analyse the results and create a Stochastic Optimality Theoretic model of object drop are available \href{https://github.com/giuliacappelli/MedinaStochasticOptimalityTheory}{on my Github profile}\sidenote{https://github.com/giuliacappelli/ MedinaStochasticOptimalityTheory} in another dedicated repository.
