\chapter{Stochastic OT models of object drop in English and Italian}
\labapp{app_models}

This appendix collects all the 18 models of the indefinite object construction that I computed for this dissertation, as detailed in \refsec{intro_models}.\\
The interested reader can browse my GitHub profile to find both \href{https://github.com/giuliacappelli/dissertationData}{the raw data}\footnote{https://github.com/giuliacappelli/dissertationData} and \href{https://github.com/giuliacappelli/MedinaStochasticOptimalityTheory}{the Python scripts}\footnote{https://github.com/giuliacappelli/MedinaStochasticOptimalityTheory} I used to compute these results.
