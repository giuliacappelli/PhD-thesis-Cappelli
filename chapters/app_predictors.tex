\chapter{Verb-dependent predictors of object drop}
\labapp{app_predictors}


This appendix collects the semantic selectivity scores, telicity feature, and manner specification of each English and Italian verb of interest. These data are also available \href{https://github.com/giuliacappelli/dissertationData}{on my GitHub profile}\footnote{https://github.com/giuliacappelli/dissertationData}. The stimuli used in the Behavioral PISA experiment are listed in the same GitHub repository and here in \refapp{app_behavPisa}.\\
The scripts used to compute the semantic selectivity scores are available on GitHub  \href{https://github.com/giuliacappelli/behavioralPISA}{here}\footnote{https://github.com/giuliacappelli/behavioralPISA} for Behavioral PISA and \href{https://github.com/ellepannitto/PISA}{here}\footnote{https://github.com/ellepannitto/PISA} for Resnik's SPS and Computational PISA.
